# noisy-cricket

A simple PHP framework.

**DEVELOPMENT IN PROGRESS**

## Main features
* Persistence
* DAO
* Friendly URLs
* Notifications
* User management and subscription
* File management

## Notes
* Created for Apache HTTP Server 2
* PHP 5.3 or higher
* It can work with other DB engines, but I've only tested with MySQL
* No classloader YET
